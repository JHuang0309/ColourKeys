# ColourKeys

A creative typing tool where each keyboard letter generates a colored block, allowing you to create visual art through typing.

🎨 **[Try it live!](https://colour-keys.vercel.app)**

## Features

- 26 unique colors (one per letter)
- 4 color palettes (Normal, Pastel, Summer, Autumn)
- Modifier keys (Shift for lighter, Cmd for desaturated)
- Adjustable canvas size
- Export as PNG
- Smooth animations
